# Global Raider Guide

A simple static website prototype for international students at Wright State University, now styled around Wright State's green-and-gold brand palette.

## Files

- `index.html` contains the page structure and content sections.
- `styles.css` contains the visual design and responsive layout.
- `script.js` powers the resource cards and search/filter behavior.

## How to use it

1. Open `index.html` in a browser.
2. Edit the `resources` array in `script.js` to add or change links.
3. Update text in `index.html` as you learn what students need most.
